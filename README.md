# Ins2-KO-Hippocampus-RNAseq
Differential expression analysis of hippocampal RNA-seq data from female Ins2-/- knockout mice (GSE305902), replicating and extending the transcriptomic findings of Baehring, O'Leary et al. 2026 (Metabologia). Includes QC/outlier detection, DESeq2 DE analysis, GO/Reactome enrichment, and GSEA with Hallmark gene sets.
